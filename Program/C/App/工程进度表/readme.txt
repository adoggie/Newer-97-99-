解决工程施工招投标过程中施工组织设计的施工进度计划表打印

//== 2000-05-25 投入使用
1、可打印分段工期
2、动态调节A3、A4的纸张宽度等
3、创建系统初始化文件，可进行参数的设置
   打印的效果比较美观！

有待改进：

  2、 根据用户要求要提供打印具体的年份月份日期
  3、 要提供打印空表
  4、 要提供比较强的数据输入的校验容错功能

注意：
   1、打印A3幅面的纸张时应注意选择的打印机和纸张类型：297 X 420
   2、 暂时不提供打印空表、多页的打印功能
